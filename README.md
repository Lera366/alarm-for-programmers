## Введение

**Будильник для программистов** — это десктопное приложение, разработанное специально для IT-специалистов, которые ценят своё время и хотят эффективно управлять рабочими интервалами.

### Основные возможности

- **Классический будильник** — установка времени срабатывания
- **Режим "Помодоро"** — таймер с циклами работы и отдыха
- **Интервальный будильник** — напоминания через заданные промежутки
- **Интеграция с календарём** — учёт встреч и задач
- **Режим "Не беспокоить"** — отключение уведомлений в заданные часы
- **Статистика продуктивности** — отчёт о соблюдении режима

---

## Примеры использования

Ниже представлены основные сценарии использования приложения:

1. **При нажатии на кнопку "Новый будильник"** открывается окно установки времени с возможностью выбора часов, минут и секунд.

2. **При выборе режима "Помодоро"** запускается стандартный цикл: 25 минут работы, затем 5 минут отдыха. После каждого четвёртого цикла отдых увеличивается до 15 минут.

3. **При нажатии на кнопку "Отложить"** будильник повторно срабатывает через 5 минут.

4. **При активации режима "Не беспокоить"** все уведомления блокируются в указанный временной интервал (например, с 23:00 до 07:00).

5. **При добавлении задачи в календарь** приложение автоматически предлагает установить напоминание за 10 минут до события.

6. **При завершении рабочего цикла в режиме "Помодоро"** появляется уведомление с предложением сделать короткий перерыв.

7. **При нажатии на кнопку "Статистика"** открывается отчёт с графиками:
   - количество завершённых циклов за день/неделю/месяц
   - средняя продолжительность рабочей сессии

8. **При выборе интервального режима** можно установить напоминания каждые 30, 45 или 60 минут для разминки или отдыха глаз.

9. **При срабатывании будильника** воспроизводится выбранный пользователем звуковой сигнал, а на экране появляется всплывающее окно.

10. **При синхронизации с системным треем** приложение сворачивается в панель уведомлений и продолжает работу в фоновом режиме.

## Технические требования

### Операционная система
- Windows 10 / 11 (64-bit)
- macOS 11 (Big Sur) и новее
- Linux (Ubuntu 20.04+, Fedora 35+)

### Аппаратные требования
| Компонент | Минимальные требования |
|-----------|------------------------|
| Процессор | 1.5 ГГц, 2 ядра |
| Оперативная память | 2 ГБ |
| Свободное место на диске | 200 МБ |
| Звуковая карта | Для воспроизведения сигналов |

### Программное обеспечение
- Python 3.9 или выше
- Библиотеки: `tkinter`, `sqlite3`, `plyer`, `schedule`
- Для работы уведомлений: системные уведомления ОС

### Дополнительные условия
- Стабильное подключение к интернету (для синхронизации календаря)
- Аккаунт Google (опционально, для интеграции с Google Calendar)
